# Roadmap

## Milestone 1: Inspectable MVP

- Local repo indexing
- Deterministic intent routing
- MCP-style tool registry
- Search/read/triage/patch proposal tools
- Event audit log
- CLI demo
- Local Web Demo
- Evaluation runner
- Rule-based monitor

## Milestone 2: Agent Workflow

- LangGraph `StateGraph`
- Optional DeepSeek answer synthesis
- Conversation memory
- SQLite persistence for sessions, turns, events, and memory
- Conditional routing
- Failure recovery node
- Human approval node for file writes

## Milestone 3: Deeper Evaluation

- 30+ end-to-end support tasks
- Retrieval hit rate
- Answer correctness rubric
- Regression reports across repository snapshots
- LLM-as-judge experiments with human spot checks

## Milestone 4: Product Surface

- FastAPI server for deployment-oriented demos
- Approval review page
- Saved conversations
- Exportable issue triage reports
- Demo screenshots or GIFs for the README
